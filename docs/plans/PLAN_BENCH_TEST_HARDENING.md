## Implementation Steps

### Step 1: Fix `_infer_category` bug (bugfix + test)

**File:** `src/hwcc/bench/scoring.py` line 425-451

**Bug:** `usb_otg_base_address` → `parts[-2]` is `"otg"`, not `"base"` → returns `"unknown"`

**Fix:** Match suffix patterns from the end, using `"_".join(parts[-2:])` for two-word categories:

```python
def _infer_category(question_id: str) -> str:
    if question_id.endswith("_base_address"):
        return "base_address"
    if question_id.endswith("_offset"):
        return "register_offset"
    if question_id.endswith("_bits"):
        return "bit_field"
    if question_id.endswith("_reset"):
        return "reset_value"
    if question_id.endswith("_access"):
        return "access_type"
    return "unknown"
```

**Tests:** `test_bench_scoring.py`
- `test_infer_category_simple_peripheral` — `spi1_base_address` → `base_address`
- `test_infer_category_underscored_peripheral` — `usb_otg_base_address` → `base_address`
- `test_infer_category_all_categories` — one test per category
- `test_infer_category_unknown` — unrecognized suffix → `"unknown"`
